# Group 4: NHIS 2021 Data Analysis Project
## Project Overview
This project provides a comprehensive analysis of the 2021 National Health Interview Survey (NHIS) data. Our goal was to manage, clean, and visualize real-world public health data to identify biological and sociodemographic trends within the U.S.

### Key Research Findings
- Physiological Correlation: Identified a statistically signifanct, moderate-to-strong positive correlation between height and weight (r = 0.502, p < 2.2e-16).
- Social Determinants: Observed a "cohort effect" where lower educational attainment was associated with a higher median age.
- Well-being: Confirmed that high life satisfaction is a strong marker for "Excellent' or "Very Good" perceived physical health.
